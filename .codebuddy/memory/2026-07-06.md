## 修复记录：bun add 损坏 pnpm node_modules

运行 `bun add recharts` 后，bun 破坏了 pnpm 的 `.pnpm` 存储结构，导致 `@code-inspector/webpack/dist/loader.js` 文件丢失。报错：
```
Error: Cannot find module '@code-inspector/webpack/dist/loader.js'
```
修复方式：`pnpm add @code-inspector/webpack@1.3.6` 重新安装。
