## Tech Stack

- Same as existing admin panel: React 19 + TypeScript + @lobehub/ui + antd + antd-style
- Mock Service: Promise-based with setTimeout delay (200-500ms), no backend dependency
- Route config: desktopRouter.config.tsx + desktopRouter.config.desktop.tsx (sync required)

## Implementation Approach

### Service Layer (Mock) Pattern

- Each module gets a class-based service (matching `adminSettingsService` pattern)
- Methods return `Promise<T>` with `setTimeout` to simulate async latency
- Mock data defined as static constants within the service file
- Comments `// TODO: Replace with real API when backend endpoints are ready` on each method
- No real API calls, so no need for `AdminApiBase` or `lambdaClient`

### UI Layer Pattern

- Reuse existing common components: `PageHeader`, `AdminSearch`, `AdminTable`, `StatusTag`
- Route files are thin: `export { default } from '@/features/Admin/Payment/PaymentSettings'`
- Form-intensive pages use antd Form with validation
- All deletion operations use Popconfirm
- All operations have loading state feedback

### Data Flow (Mock)

```
User interaction -> React UI (state change) -> Mock Service (Promise+setTimeout) -> Return mock data -> Update UI state
```

## Architecture Design

### Module Division

- **i18n**: Simple key-value JSON generation from TypeScript source
- **Payment**: Service + UI(3 Cards layout + Save) + Route + Sidebar
- **Plans**: Service + UI(CreditSettings Card + PlanList Table) + Route + Sidebar

### Data Flow (Payment)

```
PaymentSettings.tsx
  -> adminPaymentService.getPaymentConfig()  [Promise, 300ms]
  -> Admin displays 3 Card groups with form fields
  -> User edits -> Save -> adminPaymentService.updatePaymentConfig()
  -> message.success()
  // TODO: Replace with lambdaClient.config.* when backend ready
```

### Data Flow (Plans)

```
PlanList.tsx
  -> adminPlanService.listPlans() [Promise, 300ms]
  -> antd Table display
  -> Create/Edit modal -> adminPlanService.createPlan/updatePlan()
  -> Delete Popconfirm -> adminPlanService.deletePlan()
  // TODO: Replace with lambdaClient.plan.* when backend ready

CreditSettings.tsx
  -> adminPlanService.getCreditConfig() [Promise, 300ms]
  -> Form with InputNumber fields
  -> Save -> adminPlanService.updateCreditConfig()
  // TODO: Replace with lambdaClient.config.* when backend ready
```
