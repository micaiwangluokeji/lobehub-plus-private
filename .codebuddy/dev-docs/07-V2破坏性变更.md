# LobeHub 2.0 破坏性变更

> **来源**：https://lobehub.com/zh/docs/self-hosting/migration/v2/breaking-changes
> **所属**：自托管 > 迁移 > V2

## 1. 移除的环境变量

| 环境变量 | 替代方案 |
| :--- | :--- |
| `ACCESS_CODE` | 使用 Better Auth 认证系统 |
| `NEXT_PUBLIC_SERVICE_MODE` | 2.0 仅支持 Server DB 模式 |
| `NEXT_PUBLIC_ENABLE_BETTER_AUTH` | 自动检测 |
| `NEXT_PUBLIC_AUTH_URL` / `AUTH_URL` | 自动检测 |
| `NEXT_PUBLIC_ENABLE_NEXT_AUTH` | 迁移至 Better Auth |
| `NEXT_AUTH_SECRET` | 迁移至 Better Auth |
| `NEXT_AUTH_SSO_PROVIDERS` | 迁移至 Better Auth |
| `NEXTAUTH_URL` | 迁移至 Better Auth |
| `NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY` | 迁移至 Better Auth |
| `CLERK_SECRET_KEY` | 迁移至 Better Auth |

## 2. 新增的必需环境变量

- **`AUTH_SECRET`**: 用于加密会话令牌
- **`JWKS_KEY`**: 用于签名和验证 JWT

## 3. 新增的可选环境变量

| 环境变量 | 说明 |
| :--- | :--- |
| `AUTH_SSO_PROVIDERS` | SSO 提供商列表，逗号分隔 |
| `INTERNAL_JWT_EXPIRATION` | 内部 JWT 过期时间（默认 30s） |
| `AUTH_EMAIL_VERIFICATION` | 设置为 `1` 以要求邮箱验证 |
| `SMTP_HOST` | SMTP 服务器 |
| `SMTP_PORT` | SMTP 端口 |
| `SMTP_USER` | SMTP 用户名 |
| `SMTP_PASS` | SMTP 密码 |

## 4. 认证系统变更（重大变更）

- **V1**: NextAuth 和 Clerk
- **V2**: 仅支持 Better Auth

## 5. 数据库模式变更

- **V1**: Server DB + Client DB (PGlite)
- **V2**: 仅支持 Server DB 模式

## 6. PostgreSQL 版本要求

- **最低要求**: PostgreSQL 17+
- 使用 `pg_search` 插件提供全文搜索能力
- Docker 推荐镜像: `paradedb/paradedb:latest-pg17`
