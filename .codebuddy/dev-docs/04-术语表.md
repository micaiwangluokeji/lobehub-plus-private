# LobeHub 术语表

> **来源**：https://lobehub.com/zh/docs/glossary
> **所属**：LobeHub Docs

## 核心概念与实体

| 术语 | 说明 |
|------|------|
| **Agent（助理）** | 可配置的AI助理，是对话和任务分配的基本单位，可复用 |
| **Agent Group（群组）** | 多个助理协作完成任务的共享空间，支持多种协作模式 |
| **Model（模型）** | 驱动助理回复的大语言模型，如 GPT-4o、Claude |
| **Topic（话题）** | 与某个助理或群组的单次对话线程，可搜索、重命名、归档 |
| **Thread（子话题/Branching）** | 从主对话中派生的分支子对话 |
| **Workspace（工作空间）** | 指上下文侧边面板，也指拥有独立订阅和共享资源的团队空间 |

## 功能与特性

| 术语 | 说明 |
|------|------|
| **Artifact（制品）** | 助理生成的相对完整、独立的内容（网页、图表、文档） |
| **Cloud Sandbox（云沙箱）** | 助理运行代码、生成文件的安全云端环境 |
| **Command Menu（命令菜单）** | Cmd/Ctrl + K 唤起的全局搜索浮层 |
| **Image/Video Generation（图像/视频生成）** | 内置的文生图和文/图生视频工作区 |
| **Memory（记忆）** | 助理从对话中提取的个人上下文（偏好、工作习惯等） |
| **RAG（检索增强生成）** | 通过语义搜索，让助理根据知识库内容作答 |
| **Web Search（联网搜索）** | 让助理从互联网获取最新信息 |
| **Scheduled Task（定时任务）** | 按计划自动运行助理 |
| **STT / TTS** | 语音转文字 / 文字转语音 |
| **OCR（文字识别）** | 从图片和文档中读取文字 |

## 技术与开发

| 术语 | 说明 |
|------|------|
| **Embedding（嵌入向量）** | 将文本转换为数值向量，用于语义匹配和搜索 |
| **File Chunk（文件分块）** | 将大文件切分成片段并生成嵌入向量 |
| **MCP（模型上下文协议）** | 连接助理与外部工具、数据和服务的开放标准 |
| **Plugin（插件）/ Skill（技能）** | 为助理增加新工具的能力 |
| **Model Provider（模型服务商）** | 提供模型的服务，如 OpenAI、Anthropic |
| **Feature Flags（功能开关）** | 自托管部署中通过环境变量开启/关闭功能 |

## 商业模式与账户（⭐ 与管理后台相关）

| 术语 | 说明 |
|------|------|
| **Credits（额度）** | 计量AI用量的单位，来自免费额度、订阅、推荐、充值 |
| **Plan（套餐）** | 决定月度额度与功能权限的订阅档位 |
| **Budget（预算）** | 可供消耗的额度池，分为个人和工作空间两种 |
| **Subscription（订阅）** | 按月或按年计费的周期性套餐 |
| **Top-up（充值）** | 按需购买额外额度 |
| **Referral（推荐）** | 通过推荐码邀请他人注册，双方获得额度奖励 |
| **Risk Control（风控）** | 检测并拦截滥用行为的安全机制 |

## 部署与集成

| 术语 | 说明 |
|------|------|
| **Self-hosting（自托管）** | 在个人基础设施上运行开源LobeHub |
| **Channels（渠道）** | 将LobeHub助理接入外部聊天平台 |
| **Messenger** | 通过官方机器人对话 |
| **Integrations（集成）** | 助理通过MCP或OAuth与外部服务连接 |
| **LobeHub Provider** | LobeHub内置的模型服务商 |

## 关键关系

- **Skill vs Plugin**: 实质相同，Skill 是面向用户的叫法，Plugin 是内部术语
- **Thread vs Branching**: Branching 是行为，Thread 是结果（子对话）
- **Channels vs Messenger**: Channels 是第三方聊天软件集成，Messenger 是通过官方机器人对话
