## 一、概述

LobeHub 管理后台（Admin Panel）是一套完整的管理界面，供超级管理员（super_admin）管理系统资源。

### 访问地址

```
https://app.lobehub.com/admin
```

开发环境：
```
http://localhost:3010/admin
```

### 技术特性

- ✅ TypeScript 0 错误
- ✅ ESLint 0 错误
- ✅ 17 个功能模块全部完成
- ✅ 响应式设计（支持桌面端）
- ✅ 国际化支持（中文/英文）

## 二、访问权限

### 权限要求

| 角色 | 访问权限 | 说明 |
|------|---------|------|
| **super_admin** | ✅ 完全访问 | 可访问所有管理功能 |
| **vip_user** | ❌ 无权限 | 无法访问管理后台 |
| **free_user** | ❌ 无权限 | 无法访问管理后台 |

### 登录流程

1. 访问 `/admin` 路径
2. 系统自动重定向到登录页（`/signin`）
3. 使用 super_admin 账号登录
4. 登录成功后自动跳转到管理后台首页

## 三、功能模块

### 3.1 仪表盘（Dashboard）

**路径**：`/admin`

**功能**：
- 系统核心统计数据展示
- 用户数、工作区数、Agent 数等关键指标
- 最近注册用户列表
- 最近创建的工作区列表

**统计卡片**：
- 总用户数
- 总工作区数  
- 总 Agent 数
- 总消息数
- 今日新增用户
- 今日活跃用户

---

### 3.2 用户管理

**路径**：`/admin/users`

**功能**：
- 用户列表查看（支持分页）
- 用户搜索（按用户名/邮箱）
- 创建新用户
- 编辑用户信息
- 删除用户
- 角色分配

**操作说明**：

1. **搜索用户**：
   - 在搜索框输入用户名或邮箱
   - 支持模糊搜索
   - 点击"搜索"按钮或按 Enter 键

2. **创建用户**：
   - 点击"创建用户"按钮
   - 填写用户信息（用户名、邮箱、密码等）
   - 选择角色（可选）
   - 点击"确定"保存

3. **编辑用户**：
   - 点击用户行的"编辑"按钮
   - 修改用户信息
   - 点击"确定"保存

4. **分配角色**：
   - 点击用户行的"角色"按钮
   - 在弹窗中勾选/取消角色
   - 点击"保存"

5. **删除用户**：
   - 点击用户行的"删除"按钮
   - 在确认弹窗中输入用户 ID
   - 点击"确认删除"

---

### 3.3 用户详情

**路径**：`/admin/users/:id`

**功能**：
- 查看用户详细信息
- 查看用户角色列表
- 管理用户角色（添加/移除）
- 查看用户基本信息（头像、用户名、邮箱等）

---

### 3.4 角色管理

**路径**：`/admin/roles`

**功能**：
- 角色列表查看
- 创建新角色
- 编辑角色信息
- 权限绑定（为角色分配权限）
- 删除角色（仅非系统角色）

**操作说明**：

1. **创建角色**：
   - 点击"创建角色"按钮
   - 填写角色名称、显示名称、描述
   - 选择是否为系统角色
   - 点击"确定"保存

2. **权限绑定**：
   - 点击角色行的"权限"按钮
   - 在权限树中勾选/取消权限
   - 点击"保存"

3. **删除角色**：
   - 只能删除非系统角色（isSystem = false）
   - 点击"删除"按钮
   - 确认删除

---

### 3.5 权限管理

**路径**：`/admin/permissions`

**功能**：

**权限类别**：

### 3.6 工作区管理

**路径**：`/admin/workspaces`

**功能**：
- 工作区列表查看
- 查看工作区详情
- 冻结/解冻工作区
- 删除工作区
- 查看工作区成员

**操作说明**：

1. **冻结工作区**：
   - 点击工作区行的"冻结"按钮
   - 填写冻结原因（可选）

2. **解冻工作区**：
   - 点击工作区行的"解冻"按钮

3. **查看成员**：
   - 点击工作区行的"成员"按钮
   - 查看所有成员列表及其角色

## 四、操作指南

### 4.1 通用操作

#### 搜索

大多数列表页面支持搜索功能：
1. 在搜索框中输入关键词
2. 按 Enter 键或点击"搜索"按钮
3. 系统显示搜索结果

#### 分页

列表页面支持分页：
- 默认每页显示 20 条记录
- 可在页面底部切换页码
- 支持跳转到指定页码

#### 创建/编辑

创建和编辑操作通常使用 Modal 弹窗：
1. 点击"创建"或"编辑"按钮
2. 在弹窗中填写表单
3. 点击"确定"保存
4. 点击"取消"放弃操作

#### 删除

删除操作需要二次确认：
1. 点击"删除"按钮
2. 在确认弹窗中确认
3. 部分删除操作需要输入资源 ID 以防止误删

---

### 4.2 权限管理

#### 为角色分配权限

1. 进入"角色管理"页面
2. 找到目标角色，点击"权限"按钮
3. 在权限树中勾选需要的权限
4. 点击"保存"

#### 为用户分配角色

1. 进入"用户管理"页面
2. 找到目标用户，点击"角色"按钮
3. 在弹窗中勾选需要的角色
4. 点击"保存"

---

### 4.3 工作区管理

#### 冻结工作区

冻结工作区后，该工作区的所有成员将无法访问：

1. 进入"工作区管理"页面
2. 找到目标工作区，点击"冻结"按钮
3. 填写冻结原因（可选）
4. 点击"确定"

#### 查看工作区成员

1. 进入"工作区管理"页面
2. 找到目标工作区，点击"成员"按钮
3. 在侧边栏中查看所有成员及其角色

## 五、常见问题

### 5.1 无法访问管理后台

**问题**：访问 `/admin` 路径时被重定向到登录页。

**原因**：
1. 未登录
2. 登录的账号不是 super_admin 角色

**解决方法**：
1. 使用 super_admin 账号登录
2. 如果不确定是否为 super_admin，请联系系统管理员

---

### 5.2 创建用户失败

**问题**：创建用户时提示"用户名已存在"或"邮箱已存在"。

**原因**：用户名或邮箱在系统中已存在。

**解决方法**：
1. 更换用户名或邮箱
2. 如果用户已存在，可以直接搜索并编辑该用户

---

### 5.3 删除角色失败

**问题**：尝试删除角色时提示"无法删除系统角色"。

**原因**：系统角色（isSystem = true）不能被删除。

**解决方法**：
1. 只能删除自定义角色（isSystem = false）
2. 如果需要修改系统角色的权限，可以使用"权限绑定"功能

---

### 5.4 冻结工作区失败

**问题**：点击"冻结"按钮后无响应或提示错误。

**原因**：
1. 工作区已被冻结
2. 网络错误
3. 权限不足

**解决方法**：
1. 检查工作区当前状态
2. 刷新页面重试
3. 确认登录账号为 super_admin

---

### 5.5 审计日志不显示

**问题**：访问"审计日志"页面时显示"暂无数据"。

**原因**：
1. 系统中还没有任何审计日志
2. 审计日志功能未启用

**解决方法**：
1. 审计日志会在用户执行操作时自动记录
2. 如果没有数据，说明还没有任何操作被记录
3. 尝试执行一些操作（如创建用户、更新工作区等），然后再次查看

## 六、技术支持

### 6.1 日志查看

如果遇到问题，可以查看浏览器控制台日志：
1. 按 F12 打开开发者工具
2. 切换到"Console"标签页
3. 查看错误信息

### 6.2 后端 API 文档

管理后台使用以下后端 API：

**OpenAPI（REST）**：
- 基础路径：`/api/v1`
- 接口文档：`packages/openapi/src/routes/`

**tRPC**：
- 基础路径：`/trpc`
- Router 文档：`apps/server/src/routers/lambda/`

### 6.3 前端代码结构

```
src/
├── features/Admin/           # 管理后台功能模块
│   ├── Layout/              # 布局组件（侧边栏、顶部导航）
```
